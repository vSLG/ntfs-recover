# ntfs-recover

A toolkit for repairing/recovering ntfs partitions.

## Building

Clone and build:
```
$ git clone https://github.com/vSLG/ntfs-recover.git
$ cd ntfs-recover
$ make build -j$(nproc)
```

The binaries will be located at `bin` directory.
